## About This Project

This is our college project for Web Technologies Lab. We have made a Task Management System where you can add tasks, edit them, delete them, and mark them as complete. It uses HTML, CSS, and JavaScript.
The project demonstrates CRUD operations (Create, Read, Update, Delete) which we learned in class.

---

## What Does It Do?

Our application can:
- Add new tasks with details like title, description, priority, category and due date
- Show all your tasks in a nice list
- Edit any task if you want to change something
- Delete tasks you don't need anymore
- Mark tasks as completed when you finish them
- Filter tasks to see All, Pending, or Completed ones
- Show your data in XML format (for assignment requirement)
- Save all your tasks so they don't disappear when you close the browser

---

## Technologies We Used

**Frontend:**
- HTML5 - for making the structure of the webpage

### Adding a Task:
1. Fill the "Task Title" (this is required)

3. Select priority - High, Medium or Low
4. Add category like "College", "Personal" etc (optional)
5. Select due date if the task has a deadline
6. Click "Add Task" button

## Validation Rules

We added these validations in our project:
- Task title is compulsory (can't be empty)
- Title should be between 3 to 100 characters
- Due date cannot be a past date
- Description can be maximum 500 characters
- Category should only have letters and numbers

If you enter wrong data, it will show error message in red color.
